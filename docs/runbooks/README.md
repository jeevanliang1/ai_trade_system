# Runbooks

Store deployment, preview, troubleshooting, data initialization, operational recovery, and environment setup procedures here.

Do not store broad product requirements or transient local-machine notes here.

- `hourly-codex-continuation.md`: local macOS launchd task that sends `继续完成项目` to Codex every hour for this repository.
- `reviewable-commit-plan.md`: staging and verification plan for splitting the current large migration into reviewable commits.
